# Linear Watchtower

>  Your utility helper to enhance the Linear+Github workflow

## ⚙️ Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```

## 📦 Docker

```sh
# 1. Build container
docker build -t linear-watchtower .

# 2. Start container
docker run -e APP_ID=<app-id> -e PRIVATE_KEY=<pem-value> linear-watchtower
```
